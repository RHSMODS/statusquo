# LTSU-1

{% hint style="info" %}
Introduced in **0.4.1026**
{% endhint %}

Laser device that is capable of emitting visible laser of orange color and is mountable on the AKS-74U.

### Default key bind

LTSU only has an option to toggle its visible laser.
